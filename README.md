# About
Qualithon Treasure Hunt Web Application

# Requirements
 - Python 3 with pip

# Install
```
  $> pip install -r requirements.txt
```

# Run app
1. Run Treasure hunt WebApp
```
  $> python3 http-server.py
```
2. Run Websocket Service
```
  $> python3 ws-server.py
```
3. Point browser to http://localhost:5000
