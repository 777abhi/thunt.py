QTribe Test Automation Qualithon '21

A fun filled Virtual Test Automation Hackathon Competition

Qualithon is a field day organized by the Test Automation QTribe. This event requires you to use all your Automation and Hacker skills to complete a Treasure Hunt. You will have to develop a Bot that can launch and traverse the Treasure Hunt and find the hidden treasure. The Bot will have to solve puzzles at each checkpoint to find keys and unlock the next door till the treasure is found. If you want to participate in the Hackathon competition please read the instructions provided in the [LINKED] Teams channel.
